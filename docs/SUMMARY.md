## 使用说明

### 快速开始
1. **安装依赖**: `pip install pygame`
2. **启动游戏**: `python main.py` 或双击 `start-game.bat`
3. **选择模式**: 数字模式或图片模式
4. **选择难度**: 简单/中等
5. **图片选择**（图片模式）: 随机选择或指定特定图片
6. **开始游戏**: 使用鼠标点击移动方块

### 操作指南
- **移动方块**: 鼠标点击相邻方块
- **重新开始**: 点击"重新开始"按钮
- **返回菜单**: 点击"主菜单"按钮
- **语言切换**: 点击右上角语言按钮或按Alt+L
- **排行榜筛选**: 在排行榜界面点击难度按钮切换查看
- **清空排行**: 点击清空按钮后二次确认

### 自定义设置
- **添加图片**: 将图片放入`assets/images`文件夹
- **调整难度**: 在游戏中选择不同难度级别
- **查看排行**: 游戏完成后自动进入排行榜
