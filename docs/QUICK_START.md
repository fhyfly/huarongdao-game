# 华容道游戏快速开始指南

## 🚀 快速启动

### 方法一：命令行启动
```bash
# 1. 安装依赖
pip install pygame

# 2. 运行游戏
python main.py
```

### 方法二：批处理文件启动（推荐）
双击 `start-game.bat` 文件，程序会自动：
- 检查Python环境
- 安装缺失的依赖
- 启动游戏

## 🎮 游戏操作

### 基本流程
1. **选择模式**: 数字模式 或 图片模式
2. **选择难度**: 简单(3×3) / 中等(4×4) / 困难(5×5)
3. **开始游戏**: 移动方块完成拼图

### 控制方式

#### 鼠标操作
- **移动方块**: 点击想要移动的方块
- **重新开始**: 点击"重新开始"按钮
- **返回菜单**: 点击"主菜单"按钮

#### 键盘操作
| 按键 | 功能 |
|------|------|
| W/↑ | 向上移动 |
| S/↓ | 向下移动 |
| A/← | 向左移动 |
| D/→ | 向右移动 |
| R | 重新开始 |
| Q | 返回主菜单/退出 |

## 🏆 游戏特色

### 📊 实时统计
- **时间记录**: 精确到秒的完成时间
- **步数统计**: 记录总移动步数
- **成绩排行**: 自动保存最佳成绩

### 🎯 多样玩法
- **三种难度**: 适合不同水平的玩家
- **两种模式**: 数字挑战或图片拼图
- **自定义图片**: 支持个人图片导入

## ❓ 常见问题

### Q: 游戏无法启动怎么办？
A: 确保已安装Python和pygame：
```bash
pip install pygame
```

### Q: 如何添加自己的图片？
A: 将图片文件放入 `custom_images/` 文件夹

### Q: 游戏卡顿怎么办？
A: 关闭其他程序，确保有足够的系统资源

### Q: 成绩没有保存？
A: 检查是否有写入权限，`leaderboard.json` 文件是否被锁定
